# MBTI-Friend-AI
A conversational AI companion app where each of the 16 MBTI personality types is brought to life as a distinct, chatty friend — built with Streamlit and powered by GPT-4o-mini.

# Project Overview
This project explores how prompt architecture can shape AI personality at the conversation level. Rather than building a generic chatbot, each MBTI type is driven by a carefully designed system prompt that encodes tone, reasoning style, emotional range, and communication patterns — making an INTJ feel meaningfully different from an ENFP in actual conversation.
The app also includes an onboarding quiz that maps a user's current emotional and cognitive preferences to a recommended companion type, demonstrating how user context can guide AI interaction design.

# Features
16 individually prompted MBTI personality agents
- Companion-matching quiz, which includes 4 questions mapping to 4 MBTI dimensions
- Full conversation memory within each session
- Personality-aware response style with token and temperature tuning
- Deployed on Streamlit Cloud with environment-based API key management
